# Release Workflow

## When
Preparing for release.

## Steps

### 1. Review
```
@code-reviewer full review
```

### 2. Tests
```
@tester ensure all tests pass
```

### 3. Cleanup
```
@housekeeper pre-release cleanup
```

### 4. Docs
```
@docwriter update documentation
```

## Command
Just say: "@manager prepare release"
