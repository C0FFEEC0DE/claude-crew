# Bug Fix Workflow

## When
Found a bug → need to fix it.

## Steps

### 1. Explore
```
@explorer understand the bug area
```

### 2. Find
```
@bugbuster find the root cause
```

### 3. Fix
(You do the fix)

### 4. Test
```
@tester write tests for the fix
```

### 5. Review
```
@code-reviewer review the fix
```

### 6. Cleanup
```
@housekeeper cleanse
```

## Command
Just say: "@manager fix bug in [area]"
