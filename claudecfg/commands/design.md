# /design

Run design session.

## When to use
- New feature
- Architectural decision
- Don't know where to start

## Process
1. What needs to be achieved?
2. What are the constraints?
3. Solution options
4. Pros/cons of each
5. Choose and explain why

## Important
- Document decisions
- Don't over-engineer
- YAGNI — don't do for "later"
- Start simple, improve as needed
